Method

Required tabulated current-carrying capacity is estimated by applying the combined rating factor:

It = Ib ÷ C

The selected cable size is the smallest size with tabulated current-carrying capacity equal to or greater than the required value.

  • Ib = design current (A)
  • C = combined rating factor
  • It = required tabulated current-carrying capacity (A)
  • The smallest listed cable size with sufficient tabulated capacity is selected.

Reference

  • This calculator provides a preliminary cable size selection based on design current and tabulated current-carrying capacities.
  • Current-carrying capacities vary depending on cable type, insulation type, conductor material and installation method.
  • Relevant standards and guidance may include BS 7671 and associated tabulated current-carrying capacity data.
  • Final cable sizing should also consider voltage drop, earth fault loop impedance, disconnection time, short-circuit withstand and installation conditions.
Design note: This calculator is an indicative sizing tool only. Final design must check correction factors, protective devices, voltage drop, fault performance, installation method and project-specific requirements.
