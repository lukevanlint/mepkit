Reference

This calculator uses the standard LTHW relationship:

Flow Rate (L/s) = Heat Load ÷ (4.18 × ΔT)

The calculated flow rate is then checked against a material-specific pipe size table to find the smallest pipe size that keeps velocity within the selected limit.

Note: standard copper and steel pipe sizes differ, so the recommended size depends on the selected material.

Velocity guidance: the velocity commentary shown in the result is indicative only. It is based on traditional accepted design norms for closed water systems, not hard absolute rules.

Final selection should also consider pressure drop, noise, erosion risk, water quality, pipe material, manufacturer data and project standards.
