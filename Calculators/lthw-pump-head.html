Reference

This calculator estimates pump head using an assumed pipe pressure drop per metre and allowances for fittings and terminal losses.

Key relationships used:

Total Equivalent Length = Straight Length × (1 + Fittings Allowance)

Pipe Friction Loss = Equivalent Length × Pressure Drop (Pa/m)

Pump Head (m) = Total Pressure Loss (Pa) ÷ 9810

Pipe pressure drops in heating systems are commonly estimated using indicative friction rates during early-stage design.

CIBSE guidance notes that typical heating pipe design may use pressure drops in the order of 100–300 Pa/m depending on system type, pipe material and design objectives.

Reference: CIBSE Guide C – Reference Data (Section 4: Fluid Flow and Pipe Sizing) and CIBSE CPD material on heating and chilled water pipe system design.

Scope note: this calculator provides an indicative estimate only. Final pump selection should be based on the system index run and confirmed equipment losses including coils, heat exchangers, control valves and strainers.
